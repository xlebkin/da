а. git global
б. git settings
в. git config
2) Как проверить настройки параметров?

б. git config --list
в. git global --logs
3) Как инициализировать Git в папку?
а. git init
б. git create
в. git use
4) Как внести все за раз файлы в индекс?
а. git add *
б. git add
в. git index --add
5) Как скопировать чужой репазиторий?
а. git copy <Ссылка>
б. git install <Ссылка>
в. git clone <Ссылка>
